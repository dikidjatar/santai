# Santai Language

Santai is a programming language with an Indonesian syntax

# Installation

```bash
npm install -g santai-lang
```

# Usage

```bash
santai program.santai        # run a file
santai -e "spil 'Halo!'"     # eval inline
santai --help                # show help
santai --version             # show version
```
